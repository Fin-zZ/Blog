# 目录结构


``` lua
lsui
├── examples -- 组件使用案例（实际使用场景，新项目结构规范也可以参考这里）
    ── api  -- 请求方法
    ── assets -- 静态资源文件
	── directives -- 自定义指令
    ── icons 
		── svg -- svg文件
    ── mock -- mock测试数据(mock.js模拟后端接口返回数据)
    ── router -- 路由配置
    ── store -- VueX配置
    ── utils -- 工具包
    ── views -- 页面
    ── permission.js -- 全局路由权限控制
├── lib -- build生成的js
├── packages -- 封装公共包
    ── * -- 以组件或方法命名
        ── src -- 组件或方法
            ── *.vue|*.js -- 组件或方法
        ── index.js -- 导出组件，提供install方法
├── src -- 封装资源
    ── css -- 公共样式
	── directives -- 指令
	── mixins -- 混入
	── utils -- 公共js
    ── index.js -- 导出整个Lsui组件的入口文件
```
